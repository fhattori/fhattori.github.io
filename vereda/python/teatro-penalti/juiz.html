time-cobrando:

Métodos

ConduzirCobranças:

BLOCO COBRANÇA (escolha um dos times (será o time-cobrando), repita 2 vezes):

Envia mensagens SuaVez para o técnico do time e para o goleiro do outro.

Quando ambos jogadores estiverem prontos, envia mensagens CobrançaAutorizada para o bandeirinha e para os 2 jogadores.

Se o batedor fizer o gol, contabiliza 1 ponto para o time-cobrando (anotar).

Muda o time-cobrando para o outro.

Se o timeX tiver mais pontos que o outro, enviar mensagem VenceuOTime(timeX) para todos.

Se o timeY tiver mais pontos que o outro, enviar mensagem VenceuOTime(timeY) para todos.

Caso esteja empatado, repetir o BLOCO COBRANÇA até termos um vencedor.


Irregularidade:

Ignora a cobrança recém-efetuada e ordene que ela seja repetida enviando a mensagem RepitaCobrança ao técnico do time-cobrando.
