Nesta aula, nós veremos o uso de condicional. Para que serve o condicional? O condicional é uma maneira de mudar o fluxo do algoritmo dependendo de uma situação.

Imagina a situação onde você quer escrever um algoritmo que pergunte a idade de um usuário e se ele for menor de idade, exibe uma mensagem dizendo que o usuário é menor de idade.

Já sabemos como podemos perguntar alguma coisa para o usuário: 

    
    idade = input("Digite sua idade: ")



Agora precisamos transformar o que usuário digitar em um número, conforme vimos na aula anterior, para conseguir usar esse número. 

    
    idade = input("Digite sua idade: ")
    idade = int(idade)



Então, nossa variável tem um número e podemos usar o IF para compará-lo com 18 e descobrir se o usuário é menor de idade.

Se um usuário é menor, a idade dele deve ser menor que 18. 

    
idade = input("Digite sua idade: ")
idade = int(idade)
if (idade < 18):
	print("Usuário menor de idade.")

Perceba que o print está deslocado para a direita. Esse deslocamento é feito usando a tecla "Tab" ou "espaço". E esse deslocamento é o que deixa claro para o python que este comando está dentro do IF.




Agora podemos adicionar uma mensagem depois do IF dizendo boas-vindas. Mas essa mensagem deve ser exibida sempre, não importa se o usuário é ou não menor de idade. 

    
idade = input("Digite sua idade: ")
idade = int(idade)
if (idade < 18):
 print("Usuário menor de idade.")
print("Boas-vindas!")

Note que o segundo print não tem o espaço no começo, para deixar claro que ele está fora do IF. E, portanto, será executado independentemente da idade do usuário.




Mas e se quisermos colocar uma mensagem caso o usuário seja maior de idade também? Devo adicionar mais um IF depois desse?

Na verdade, existe uma maneira mais simples: else. O else em inglês significa "senão". Usamos o else logo depois de um if e os comandos do else serão executados caso a condição do if seja falsa. 

    
idade = input("Digite sua idade: ")
idade = int(idade)
if (idade < 18):
 print("Usuário menor de idade.")
else:
 print("Usuário maior de idade.")
print("Boas-vindas!")
