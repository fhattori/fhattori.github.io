- O Python tem 2 diferentes opções para criar um loop.

A primeira opção é o loop que a gente chama de while.

"While" em inglês significa "enquanto". O loop de while cria uma condição e enquanto esta condição for verdadeira, o loop se repetirá (ou seja, todos os comandos dentro do while será executados novamente).

5. (corrigindo o algoritmo) O resultado do algoritmo anterior será 54
Isso acontece, porque a função input() vai pegar o que o usuário escreve como se fosse um texto.
Para fazermos contas matemáticas, precisamos antes transformar este texto em número. E fazemos isto utilizando a função int() ou a função float()
Então, o algoritmo correto da questão 5 deveria ser: 

    
      a = input("Digite o valor de a: ")
      b = input("Digite o valor de b: ")
      a = int(a)
      b = int(b)
      resposta = a+b
      print(resposta)
