Diálogos no scratch

  • Abra o este projeto do Scratch: Piada de Pinguins
  • Usando o ipad e procurando as palavras no google. Vamos tentar entender a piada que o pinguim está contando nessa história.
  • Depois de ver a história e entender a piada, nós vamos criar o nosso próprio diálogo no Scratch.
  • Abra o interior desse projeto do Scratch para você enteder como o diálogo foi feito e como as falas dos personagens foram sincronizadas.
  • Agora, pense em uma boa piada e escreva no Scratch da mesma maneira que esse projeto que vimos. Crie 2 personagens e 1 personagem deve contar a piada para o outro.
  • Utilize a mesma maneira de sincronizar o diálogo entre os personagens.
