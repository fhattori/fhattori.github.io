Projeto Final

Criação do nosso projeto Final

Primeiro vocês devem se separar em grupos.
Depois, escolham no grupo uma das matérias que vocês tiveram este ano na escola.
Vocês vão fazer um projeto no Scratch que tenha como tema essa matéria que vocês escolheram.

Uma boa ideia para começar a fazer este projeto é utilizar algum dos jogos que já fizemos durante este ano.
Analisem os jogos que aprendemos a fazer durante o ano.

  • Scratch - jogo Pega-pega
  • Scratch - jogo Pong
  • Scratch - jogo Obstáculos
  • Scratch - Bichinho Virtual
  • Scratch - Jogo Catch
  • Scratch - Quiz

Vocês podem também escolher fazer uma animação, mas para isto, precisaram incluir tanto um diálogo quanto mudanças de cena;

  • Scratch - Diálogo
  • Scratch - Mudança de cena

Para a primeira aula, nós vamos nos preocupar em analisar cada um desses jogos e discutir em grupo sobre os jogos e sobre a matéria que vocês escolheram.
Pensem em como vocês podem mudar o jogo para ele ensinar alguma coisa dessa matéria.

Depois anotar em papel as seguintes informações:

  • Nomes dos membros do grupo;
  • Nome da pessoa responsável pelo Scratch: esta pessoa é responsável por fazer o login no Scratch onde vocês faram o seu projeto. Escolham uma pessoa confiável, que falte pouco e que será capaz de lembrar seu login e senha na próxima aula.;
  • Projeto que vocês escolheram fazer (jogo ou animação?);
  • Matéria escolhida para o projeto;
  • Descrição do projeto: como vocês vão adaptá-lo para a matéria que vocês escolheram?;

Na próxima aula, vamos começar a fazer nosso projeto.
